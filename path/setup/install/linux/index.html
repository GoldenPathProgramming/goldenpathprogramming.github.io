Installing GCC, CMake, GDB, and Git on Linux

Arch (SteamOS, CachyOS, Omarchy, EndeavourOS): sudo pacman -S base-devel cmake gdb git libx11

Debian (Ubuntu, Mint): sudo apt install gcc-15 cmake gdb git libx11-dev

For other distros, please search for a Community guide.

Check your installation with the following commands (press enter after each one and check the output):

gcc --version cmake --version gdb --version git --version
